## Download | 下载项目

```bash
$ git clone https://github.com/ElecRex/HospitalManagementSystem-WPF.git
$ cd HospitalManagementSystem-WPF
```

## Requirements

- Visual Studio 2017

- .NET Framework 4.6.1

- SQL Server 2012 / MSSQL LocalDB

## Architecture | 系统架构

```
|__HospitalInfomationManagementSystem
  |
  |__ClassLiberty
  |         |_____Model
  |         |_____DAL
  |         |_____BLL
  |__UI
  |
  |__Database
```
